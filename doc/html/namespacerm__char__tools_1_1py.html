Detailed Description

This is a little tool for creating and keeping track of Role Master Characters.

This tool handles

  • Creation of new RM characters
  • Creation and keeping track of RM character parties to select specific information to GM
  • level development of characters (simple character history included)
  • Calculation of EPs
  • Export characters/party infomation as JSON, LaTex, PDF
  • Buying/keeping track of character's equipment & treasures
