Detailed Description

Rolemaster specific toolbox.

This package holds RM specific tools like Charakter Skill Progression.

This function reads the ProfBonus.csv and delivers a structured dictionary for further computation.

Parameters

Return values

  • 2.lvl: Prime Stats (list), Profession Bonusses (dict, 3. lvl), Realm (string)
