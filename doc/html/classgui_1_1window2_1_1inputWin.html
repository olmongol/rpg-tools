Detailed Description

Objects of this class type are windows for input the wanted data structure.

A XML structure will be build of the input.

Parameters
langThis parameter holds the language chosen for the menus and messages. Default value is 'en'.
xmlcontenta dictionary holding the XML structure/tags
filenamethis holds the filename of a read XML file holding the functional structure.
storepaththe path where the XML files shall be stored in.
