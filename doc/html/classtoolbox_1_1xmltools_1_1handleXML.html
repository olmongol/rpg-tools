Detailed Description

This class offers methods to convert dictionaries to XML and back.

It also is able to read XML-Files or save them.

Parameters
langsupported language for information
xmlstructXML structure in a dictionary.
filenameName of the file where the XML data shall be read from or written to.
linkfilename of a linked structure XML file.
actiontype of the IO action: read or write.
